# CakePHP SevenNotification Plugin

The **SevenNotification** plugin allows you to send SMS notifications via [Seven.io](https://www.seven.io) (formerly SMS77) using the CakePHP Notification plugin. Seven.io is a reliable SMS gateway provider offering global SMS delivery with high delivery rates.

The plugin provides simple SMS sending via the official Seven.io SDK, a fluent message builder API, flash SMS support for urgent messages, delayed/scheduled sending, custom sender names, and message tracking with analytics. It integrates seamlessly with CakePHP's Notification system.

The plugin uses the official Seven.io SDK, provides full type safety with PHPStan Level 8 compliance, and includes comprehensive test coverage.

## Requirements

* PHP 8.2+
* CakePHP Notification Plugin
* Seven.io Account & API Key

See [Versions.md](docs/Versions.md) for the supported CakePHP versions.
